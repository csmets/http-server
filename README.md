# http-server
Basic vanilla server written in C

## Defaults

- Default port is `8080`
- Web files are served at `/var/www/html`
- Log is found at `/var/log/http_access.log`

## Building build:

```
make build
```

Rebuilding

```
make rebuild
```

Cleaning the build

```
make clean
```

Create debug version

```
make debug
```
